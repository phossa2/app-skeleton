# phossa2/app-skeleton

**phossa2/app-skeleton** is a phossa2 application skeleton.

Installation
---
Install via the `composer` utility.

```bash
composer require "phossa2/app-skeleton"
```

or add the following lines to your `composer.json`

```json
{
    "require": {
       "phossa2/app-skeleton": "*"
    }
}
```
